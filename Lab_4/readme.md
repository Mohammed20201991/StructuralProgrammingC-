# Student Marks Analyzer

**Simple student marks analyzer. It reads marks for a group of students and computes statistics and grades.**

## Features

### Input
- Enter number of students `n`
- Input `n` student marks

### Processing
While reading marks, the program calculates:
-  Total sum of marks  
-  Highest mark (max)  
-  Lowest mark (min)  
-  Number of passed students (marks ≥ 50)

### Output
- Average marks  
- Highest mark  
- Lowest mark  
- Number of passed students  

## Formula

- **Average Marks = sum / n**


## Grading System

| Marks Range | Grade |
|------------|-------|
| 90+        | A     |
| 80–89      | B     |
| 70–79      | C     |
| 60–69      | D     |
| 50–59      | E     |
| < 50       | F     |

## Example

### Input
```
Enter number of students: 5
Enter marks:
85
92
47
60
73
```

## Output:
Average Marks = 71.4

Highest Mark = 92

Lowest Mark = 47

Grades:
